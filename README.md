# OSINT Training Portfolio

Welcome to my OSINT Training Portfolio.

This repository documents Open-Source Intelligence (OSINT) exercises completed as part of my ongoing professional development. The purpose of these reports is to demonstrate structured investigative methodology, analytical thinking, evidence collection, source validation, and professional report writing using publicly available information.

Each investigation follows a consistent workflow designed to mirror real-world intelligence and investigative practices while adhering to ethical and legal standards.

## Objectives

* Develop repeatable OSINT investigation methodologies
* Improve analytical reasoning and documentation skills
* Practice geolocation, SOCMINT, reverse image analysis, metadata analysis, and infrastructure research
* Build a professional portfolio showcasing investigative capabilities
